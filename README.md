### Запуск приложения
Для запуска приложения у вас должен быть установлен докер.

```shell
make dc_build
make dc_up
```
```shell
make app_install
```
### Docker-container
```shell
make app_bash
```

### Остановка приложения
```shell
make dc_stop
```

### Документация swagger доступна по адресу
```shell
http://localhost:8080/api/documentation
```
